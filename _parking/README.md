# _parking

Asset parking lot: "maybe use later" art, mockups, previews, and cut/regen
experiments. Nothing here is referenced by the live site or the game.

Subfolders:
- `logos/`     alternate logo cuts & banners
- `ui/`        UI chrome mockups, panel/plate/medallion samples, icon tests
- `creatures/` cut/regen/despill creature image experiments
- `floors/`    floor plate mockups & empty grids

Consolidated from the former `_previews/`, `codex/images/review/`, and
`floors/previews/` folders.
