EGGS

Hatching Mechanics

Every creature has its own egg. Hatching an egg always produces that specific creature — a Bristle Egg hatches a Bristle, a Nightmare Egg hatches a Nightmare. There are as many eggs as creatures in the tower.

Egg artwork is shared: an egg's shell shows the creature's first type, so every creature whose first type is Beast uses the Beast egg image, and so on. The image is cosmetic — it does not change what hatches.

Eggs drop at a grade on the shared 10-tier scale used across the game (Worn → Common → Uncommon → Rare → Epic → Legendary → Abyssal → Divine → Primordial → Void). Each grade of egg has a chance to hatch a creature of each grade — the egg's grade does not fix the hatchling's grade, it weights the roll: higher-grade eggs give higher chances of higher-grade creatures. Exact weights live in balance tuning.

Egg Types

The shared egg artwork, one image per first type. Every creature of that first type uses this image for its own egg. Hover a medallion for the type name.
